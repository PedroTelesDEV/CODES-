Sistema Bibliotecário em Python

Este projeto é um **sistema bibliotecário simples**, desenvolvido em Python, que utiliza arquivos **JSON** para armazenar dados de livros, usuários e empréstimos.  
O objetivo é simular o funcionamento básico de uma biblioteca, permitindo cadastro, consulta e controle de empréstimos de livros.


Funcionalidades:
- Cadastrar e listar livros
- Cadastrar e listar usuários
- Registrar e devolver empréstimos
- Armazenar dados localmente em arquivos `.json`
